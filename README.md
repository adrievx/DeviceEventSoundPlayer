# DeviceEventSoundPlayer
Fix for Classic7's device modified event sounds (connect/disconnect)

## Installation
Simply copy the executable to any local path (e.g. `C:\Applications\DeviceEventSoundPlayer.exe`) and add a shortcut in Startup or use [nssm](https://web.archive.org/web/20240303000601/https://nssm.cc/download) to install it as a service
